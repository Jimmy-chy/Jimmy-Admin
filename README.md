# 一个基于Vue3和Element-Plus的通用管理系统
主要用于熟悉Vue3、Element UI，以及使用Echarts画图

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
